# VistaUpdater
VistaUpdaterおよびWin7Updaterのソースコードです。
このレポジトリには、「VistaUpdater」「Win7Updater」「VistaUpdater 修復ツール」を含みます。
## ビルドの必須要件
- Visual Studio 2022 (.NET デスクトップ開発のワークロード)
- .NET Framework v3.0 (Windowsの機能からインストール)
